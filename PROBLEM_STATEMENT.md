# PROBLEM STATEMENT

## Title

Basic Calculator using C++

---

# Problem Statement

Develop a console-based calculator application using C++ programming language that can perform basic arithmetic operations such as:

* Addition
* Subtraction
* Multiplication
* Division

The program should:

* Accept user input
* Display accurate results
* Handle invalid operations
* Prevent division-by-zero errors
* Provide a user-friendly interface

The application should allow users to perform multiple calculations during a single execution.

---

# Purpose of the Project

The purpose of this project is to understand the fundamentals of C++ programming and implement arithmetic logic using programming concepts like:

* Variables
* Loops
* Conditional Statements
* Switch Case
* Input/Output Operations
