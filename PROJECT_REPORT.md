# PROJECT REPORT

## Title

Basic Calculator using C++

---

# 1. Introduction

The Basic Calculator is a console-based application developed using C++.
The purpose of this project is to perform simple arithmetic calculations in an efficient and user-friendly manner.

This project helps beginners understand important programming concepts such as:

* Variables
* Conditional statements
* Loops
* User interaction
* Error handling

---

# 2. Objective

The main objectives of the project are:

* To develop a calculator using C++
* To perform basic arithmetic operations
* To understand switch-case implementation
* To improve problem-solving skills

---

# 3. Software Requirements

* C++ Compiler
* VS Code / Code::Blocks
* GitHub Account

---

# 4. Functionalities

The calculator can perform:

1. Addition
2. Subtraction
3. Multiplication
4. Division

The application also:

* Prevents division by zero
* Allows repeated calculations
* Displays results clearly

---

# 5. Algorithm

1. Start the program
2. Take two numbers as input
3. Take operator input
4. Perform selected operation
5. Display result
6. Ask user whether to continue
7. Stop if user selects exit

---

# 6. Output

The calculator displays the calculated result based on user input.

Example:

```text
Input:
10 + 5

Output:
15
```

---

# 7. Conclusion

The Basic Calculator project successfully demonstrates the implementation of arithmetic operations using C++.
This project enhances understanding of programming fundamentals and improves logical thinking.
