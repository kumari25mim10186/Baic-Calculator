# Basic Calculator using C++

## 📌 Project Overview

The **Basic Calculator** is a simple C++ console-based application developed to perform arithmetic operations efficiently.
This project demonstrates fundamental programming concepts such as:

* User Input Handling
* Conditional Statements
* Switch Case
* Loops
* Error Handling
* Arithmetic Operations

The calculator provides a user-friendly interface where users can repeatedly perform calculations until they choose to exit.

---

# 🎯 Objective

The objective of this project is to build a calculator application using C++ that can:

* Accept user input
* Perform mathematical calculations
* Display accurate results
* Handle invalid operations professionally

# ▶ How to Run the Project

## Step 1: Compile the Program

```bash
g++ code.cpp -o calculator
```

## Step 2: Run the Program

### Windows

```bash
calculator.exe
```

### Linux/Mac

```bash
./calculator
```
